### Chapter: Session in Play
**Priority:** High  
**Estimated Effort:** Large

Create a comprehensive "Session in Play" chapter showing how different scene types flow together in actual gameplay. This chapter would demonstrate:

**Scene Types to Include:**
1. **Exploration** - Navigating dangerous terrain, discovering clues, overcoming environmental obstacles
2. **Social Conflict** - Negotiating with NPCs, using Will tracks, Talk vs Clash distinction
3. **Investigation** - Using Sense and Know to solve mysteries, piece together clues
4. **Combat** - Full tactical encounter with Complications, Challenge Moves, and group dynamics
5. **Downtime** - Short/long rests, recovering Stress/Conditions, preparing for next adventure

**Structure:**
- **Option A:** Linear session play-through following one party through all scene types

- **Option C (Recommended):** Hybrid - brief narrative thread connecting all scenes + detailed breakdowns with GM commentary

**Key Elements to Showcase:**
- Scene transitions and pacing
- GM telegraphing stakes before rolls

- How Momentum flows between scenes
- How Stress/Conditions carry forward and affect later scenes
- Short vs. long rest timing and recovery
- GM thought process and decision-making
- Player agency and creative problem-solving

**Sidebar Callouts:**
- "Why this is a roll" vs "Why this isn't a roll"
- "How to telegraph stakes"

**Target Audience:** New GMs learning how to run the system, experienced GMs transitioning from other systems

### Scene-Long Debuff Counter-Play Guidance
**Priority:** Medium  
**Estimated Effort:** Small

Add GM guidance for handling scene-long debuffs (Potent spells, conditions) against Tier 3-4 boss enemies to prevent "land one debuff, coast to victory" scenarios.

**Suggested approaches:**
- Boss spends turn dramatically breaking the effect
- Impact Move: "Dragon's rage burns through your curse" (remove debuff + create danger)
- Soft Move: Weaken from scene-long to "2 actions remaining"
- Boss phases: Each phase sheds debuffs (keeps fight dynamic)

**Location Options:**
- Chapter 6 (Challenges) - GM Advice section
